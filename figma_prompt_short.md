# Промпт для Figma Make: Веб-приложение перевозки автомобилей

## Проект
Веб-приложение для ООО «КИБЕРТРАКС» — перевозка премиальных автомобилей. Современный премиальный дизайн, две части: пользовательская (заказ) и административная (управление).

## Стиль
Современный минимализм, премиальный вид. Цвета: темно-синий/черный основной, золотой/оранжевый акцент. Адаптивность: десктоп и планшет. Использование Tailwind CSS.

## ЭКРАНЫ (12 штук)

### ПОЛЬЗОВАТЕЛЬСКАЯ ЧАСТЬ

**1. Главная страница**
- Шапка с логотипом "КИБЕРТРАКС", меню
- Hero-секция: заголовок о перевозке премиальных авто, CTA "Заказать перевозку"
- Блок преимуществ (закрытые автовозы, безопасность)
- Популярные маршруты (Бишкек-Москва, Москва-Сочи)
- Футер

**2. Регистрация/Авторизация**
- Форма: ФИО, Телефон (маска)
- Переключение регистрация/вход
- Валидация полей

**3. Форма заказа**
- Марка/модель авто: текстовое поле + выпадающий список (BMW, Mercedes, Audi, Lexus, Porsche)
- Город отправления: выпадающий список с поиском
- Город назначения: выпадающий список с поиском
- Дата начала: календарь
- Кнопка "Рассчитать стоимость"

**4. Расчет стоимости**
- Информация о заказе (авто, маршрут, дата)
- Расстояние (км), тариф (руб/км)
- Стоимость перевозки (крупно)
- Страховка (10%)
- **Итого к оплате** (акцент)
- Время в пути (часы/дни), дата прибытия
- Кнопки: "Подтвердить заказ", "Изменить параметры"

**5. Подтверждение заказа**
- Карточка с полной информацией (клиент, авто, маршрут, даты, стоимость)
- Чекбокс "Согласен с условиями"
- Кнопка "Перейти к оплате"

**6. Оплата (симулятор)**
- Сумма к оплате (крупно)
- Форма карты:
  - Номер карты (16 цифр, пробелы каждые 4)
  - Срок действия (MM/YY)
  - CVV (3 цифры)
  - Имя держателя
- Автоопределение платежной системы (логотип Visa/Mastercard/МИР по первой цифре)
- Кнопка "Оплатить" (крупная)
- Индикатор безопасности

**7. Успешная оплата**
- Иконка успеха (галочка, зеленый)
- "Оплата успешно выполнена"
- Номер заказа
- Кнопка "Вернуться на главную"

---

### АДМИНИСТРАТИВНАЯ ЧАСТЬ

**8. Авторизация админа**
- Форма: Логин, Пароль
- Кнопка "Войти"
- Заголовок "Панель администратора"

**9. Реестр заказов**
- Навигация: Реестр / Города / Тарифы / Выйти
- Фильтры:
  - Дата (календарь)
  - Период (диапазон)
  - Город отправления/назначения
  - Статус оплаты (Все/Оплачено/Не оплачено)
- Сортировка: по дате, по стоимости
- Таблица заказов:
  - Колонки: №, Клиент, Авто, Маршрут, Дата начала, Дата прибытия, Стоимость, Статус оплаты, Статус перевозки
  - Цвет строк:
    - **Зеленый**: не уехал (дата начала > выбранной даты)
    - **Желтый**: в пути (дата начала ≤ выбранной даты ≤ дата прибытия)
    - **Синий**: прибыл (дата прибытия < выбранной даты)
  - Клик по строке → детали
- Пагинация

**10. Детали заказа**
- Кнопка "Назад"
- Заголовок "Заказ №XXX"
- Карточка: клиент (ФИО, телефон), авто, маршрут, даты, стоимость, статусы
- Изменение статуса оплаты (выпадающий список)
- Кнопка "Сохранить"
- Цветной бейдж статуса перевозки

**11. Управление городами**
- Заголовок "Управление городами"
- Кнопка "Добавить город" (+)
- Таблица: Название, Действия (Редактировать/Удалить)
- Модальное окно: Поле "Название", Кнопки "Сохранить"/"Отмена"
- Подтверждение удаления

**12. Управление тарифами**
- Заголовок "Управление тарифами"
- Таблица: Месяц, Тариф ≤1000 км, Тариф >1000 км, Действия
- Модальное окно редактирования: Месяц, Тариф ≤1000 км, Тариф >1000 км
- Блок фиксированных маршрутов:
  - Москва-Сочи: 200 000 руб
  - Бишкек-Москва: 350 000 руб

## ТРЕБОВАНИЯ

### Цвета
- Основной: темно-синий (#1E3A5F) или черный
- Акцент: золотой (#D4AF37) или оранжевый
- Успех: зеленый (#10B981)
- Предупреждение: желтый (#F59E0B)
- Информация: синий (#3B82F6)
- Ошибка: красный (#EF4444)

### Компоненты
- Формы с плавающими лейблами
- Кнопки: primary, secondary, danger
- Карточки для группировки
- Модальные окна
- Таблицы с сортировкой
- Цветовые бейджи статусов
- Иконки минималистичные

### Интерактивность
- Hover-эффекты
- Плавные переходы
- Индикаторы загрузки
- Валидация форм (красная рамка при ошибке)

### Адаптивность
- Десктоп: 1920px, 1440px, 1280px
- Планшет: 768px, 1024px
